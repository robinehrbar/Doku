Internet

Als allererstes, bevor man mit dem Scripten anfängt sollte man überhaupt wissen wie das Internet funktioniert, um später alle Zusammenhänge verstehen zu können. Das Internet ist eigentlich nichts anderes als ein sehr grosses Netzwerk, wie jedes andere Netzwerk verbindet das sogenannnte World Wide Web (www) Computer und andere Geräte miteinander. Ohne das Internet wäre die heutige Welt - so wie sie ist - unvorstellbar. Das Internet arbeitet mit IP Adressen, um alle Geräte identifizieren und um mit denen kommunizieren zu können.

HTML

HTML kurz für HyperText Markup Language, bezeichnet die heutige standard Markup Sprache für Websiten und deren Programme. HTML beschreibt die Struktur einer Website und verwaltet damit ihr Erscheinungsbild und Aussehen. Bei HTML redet man grundsätzlich von Elemten, und eben nicht wie Viele denken von Befehlen. Diese Elemente bestehen immer aus Tags. Entweder bestehen die Elemente aus einem einzigen Tag(leeres Element sprich ohne Inhalt) oder aus einem öffnendem und einem schliessendem Tag(mit Inhalt). In den Tags selbst hat es teilweise Attribute, diese bestimmen gewisse Eigenschaften über ein Element.

Struktur

Im Prinzip funktioniert die Strukturierung ähnlich wie bei dem uns allen bekannten Microsoft Word. Nämlich hat man verschiedene Formate für jeweils eine Überschrift oder bestimmten Absatz. Zusätzlich unterteilt man bei HTML noch in header, body und footer, um später einfacher formatieren zu können.

Text

Bei HTML gibt es 6 verschiedene Überschriften, nämlich von h1 bis h6, wobei h1 die Grösste und h6 folglich die Kleinste ist. Die normalen Absätze werden zwischen p tags geschrieben. Ausserdem werden sie mit b fett, mit i kursiv, mit del durchgestrichen, mit ins unterstrichen, mit sup hochgestellt und und mit sub tiefgestellt. Ausserdem kann man mit br einen Absatz einfügen, das Gleiche gilt für hr, nur der Unterschied ist, dass bei hr noch eine Linie dazwischen ist. Auch interessant ist abbr, damit kann man nämlich beim hovern über dem Text einen kleinen eingeblendeten Text sehen.

Listen

Grundsätzlich werden Listeneinträge mit li gemacht, diese Einträge sind dann entweder in geordneten Listen(ol), ungeordneten Listen(ul) oder in Defintionslisten(dl)

Links

Links sind typische Elemente die ein Attribut brauchen, in dem a Element wird ein href Attribut mit einer Website versehen, als Elementinhalt schreibt man dann einfach das was man anklicken soll, et voilà der Link ist fertig. Diese Links können entweder auf einen verwandten Ordner(relativer Link) oder auf eine komplette Adresse, zum Beispiel https://www.google.com verweisen(absoluter Link). Fügt man dem a Element noch ein target="_blank" Attribut hinzu, wird der Link in einem neuen Tab geöffnet.

Bilder

Ähnlich wie bei den Links, referieren auch die Bilder auf absolute oder relative Links, jedoch ist es hier ein img Element anstelle des a Elements bei den Links. Mit alt="Beispieltext" kann zusätzlich noch ein Infotext zu dem Bild hinzugefügt werden, welcher auch zur Suchmaschinenoptimierung hilft. Mit height und width kann Höhe und Breite des Bildes in Pixel bestimmt werden. Am besten eignen sich png, jpg oder auch gif als Bildformate.

Video & Audio

Hier ist es auch nicht so unterschiedlich zu Bilder / Links, ausser dem Üblichen kann man hier noch zusätzlich autoplay, loop, controls und preload einstellen. Autoplay lässt das Video automatisch abspielen, Loop zwingt eine automatische Wiederholung auf, Controls fügt browserspezifische Steuerungselemente hinzu und Preload lässt das Video schon im vorhinein laden.

CSS

CSS wendet Formatierungsregeln auf HTML-Elemente an, sie legen dadurch fest, wie der Inhalt der Elemente angezeigt werden soll. Eine CSS-Regel besteht aus zwei Teilen: einem Selektor und dessen Deklaration. Selektoren geben an, auf welches Element/Klasse/Id die Regel angewandt werden soll. Deklarationen hingegen bestimmen wie die Elemente formatiert werden sollen.

CSS ins HTML einbinden

Man sollte immer ein externes CSS Sheet verwenden, um das Ganze besser zu unterteilen und Verwechslungen zu vermeiden.
